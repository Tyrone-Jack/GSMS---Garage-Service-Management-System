TROUBLESHOOTING QUICK REFERENCE
================================================================================

PROBLEM: npm is not found
SOLUTION: Install Node.js from https://nodejs.org/

PROBLEM: Port 3000 is in use
SOLUTION: Kill other processes or change port in server.js

PROBLEM: Cannot GET /login.html
SOLUTION: Make sure server is running (check terminal for errors)

PROBLEM: Cannot login
SOLUTION: Use exact credentials above (case-sensitive)

PROBLEM: Data not loading
SOLUTION: Delete gsms.db file and restart server

For more issues, see TROUBLESHOOTING.md

PROJECT FILES (WHAT EACH DOES)
================================================================================

server.js
    → The backend server (Express.js)
    → Handles all API requests
    → Manages database connections
    → Serves static files

database.js
    → SQLite database setup
    → Creates tables on startup
    → Seeds demo data
    → Defines database schema

index.html
    → Landing page with features
    → Links to login and demo info
    → Shows role buttons

login.html
    → User login page
    → Validates credentials via API
    → Redirects to appropriate dashboard

admin-dashboard.html
    → Admin interface
    → Shows booking requests
    → Allows approving/rejecting jobs
    → Lists technicians

tech-dashboard.html
    → Technician interface
    → Shows assigned jobs
    → Allows updating job status
    → Tracks work progress

customer-dashboard.html
    → Customer interface
    → Shows active jobs
    → Displays invoices
    → Allows booking services

book-service.html
    → Service booking form
    → Allows selecting vehicle and service
    → Submits booking to backend

styles.css
    → All styling for the application
    → Orange color theme
    → Responsive design
    → Professional appearance
